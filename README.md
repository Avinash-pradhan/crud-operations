# Django CRUD Application

This is a simple Django-based CRUD (Create, Read, Update, Delete) web application.

## Features
- Create new records
- View records
- Update existing data
- Delete records

## Tech Stack
- Python
- Django
- SQLite

## How to Run
1. Clone the repository
2. Install dependencies
3. Run server:
   python manage.py runserver
